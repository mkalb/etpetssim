# ET Pets - Roadmap

> Future ideas and enhancements beyond V1.

## Terrain

- Better terrain generation (non-random clustering, biomes, organic patterns)
- Obstacle clustering strategies
- Terrain compatibility per resource type

## Resources

- Water-based resources (fish or water plants)
- Per-trait mutation rates for resource regeneration

## Agents

- Multiple species
- Optional sex/gender model
- Separate `currentHealth` model
- Social behavior (avoidance, grouping, territory)
- Advanced sensing (memory, scent, probabilistic vision)
- Line-of-sight blocking by rock
- Population soft/hard cap options
- Reproduction history tracking
- Per-trait mutation rates
- Rework pet color adjustment in main view

## Genetics

- Mendelian-like genetics model
- Dynamic ecosystem balancing

## UI

- Enhanced observation view (info panel for selected cells)
- User actions to add/remove entities (Pet, Rock, Plant, Insect, etc.)
- Lineage visualization and analytics dashboard
- Scenario presets and config profiles

## Product Features

- Replay/time-travel
- Event system (disasters, seasonal shifts)
